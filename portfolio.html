Grubbr

Source Code

Grubbr is an alternative to Yelp - it works as Tinder but for food: it will match you with the best dish in town. It also allows to search for best menu items in restaurants, or simply look for specific food but ensures to be the best item by app's users. I created front-end architecture using React Native for its ability to expand on multiple interfaces without any restrictions. I applied Native Base library to enhance the rendering logic, allowing polished integration of new features like deck swipe. I designed the UI and app's artistic elements (the Fork Monster) with Pixelmator, and used CSS to improve app's overall looks and quality. Our team chose to use Redux to ensure the complicated flow of data, allowing the incorruptible transition of fetched data. For back-end we use Node and Express. For database we chose PostgreSQL because our data system is very complicated and required a relational data system for smooth and quick data retrieval. Nodal helped to quicken the database build process. The app is built for iOS and Android and I was responsible for integration of iOS features to esnure a better and uncomplicated user experience.

Pick Up is a legacy project, which originaly was created for helping to pick a team to play soccer on a certain time.
